API Endpoints

POST /api/json Download video stream URL
POST /api/formats List all available qualities
POST /api/audio Extract audio-only stream
POST /api/image Extract thumbnails
POST /api/info Get media metadata
POST /api/key Generate an API key

Example (cURL)

curl -X POST https://your-server.com/api/json \
  -H "Content-Type: application/json" \
  -H "X-API-Key: dp_your_key_here" \
  -d '{"url":"https://youtube.com/watch?v=...","videoQuality":"720p"}'
